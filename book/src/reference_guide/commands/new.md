# New

## Description

Create a new project directory containing the minimal configuration files to run a first build.

``` shell
my_project/
├── dataset.toml
├── doc
├── include
├── module.toml
├── README.md
├── src
└── target.toml
```

## Synopsys

`hydra` `new` [*option*] *name*

## Options

`--help`: Display command description and options.

## Examples

`hydra` `new` my_project
